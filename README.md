# homebrew-spelunk

Homebrew tap for [spelunk](https://github.com/spelunk-cloud/spelunk), code
intelligence for AI agents.

## Install

```bash
brew install spelunk-cloud/spelunk/spelunk
spelunk --version
```

## Maintenance

`Formula/spelunk.rb` is generated and pushed automatically by the
`update-homebrew-formula` job in
[`spelunk-cloud/spelunk`'s release workflow](https://github.com/spelunk-cloud/spelunk/blob/main/.github/workflows/release.yml)
on every stable release tag. Do not hand-edit `Formula/spelunk.rb` — changes
will be overwritten on the next release.
